# Calculadora Web

## Descrição

Projeto desenvolvido para a disciplina de Práticas Profissionais I do curso de Análise e Desenvolvimento de Sistemas.

A aplicação consiste em uma calculadora web capaz de realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar operações
- Apagar último caractere

## Objetivo

Aplicar conhecimentos básicos de desenvolvimento web na criação de uma aplicação funcional e organizada.
